# RAG

- Function Call 给 llm 提供工具和服务
- mcp 定义了 llm 与外部资源的通信协议 
RAG（检索增强生成）是一种通过从外部知识库检索相关信息并将其融入提示词，来增强大语言模型生成内容的准确性、时效性和相关性的技术框架。

- 增强？
   给 LLM 提供丰富的上下文 
- 检索？
    律师、ppt、code 知识库 

    prompt -> 知识库（检索embedding）-> prompt + 知识库（相关内容）-> LLM 返回结果 

- 如何做检索？
  在知识库中根据prompt 找到相关的那一段内容，交给大模型

  - embedding 向量
    把文本转换为向量
  - 向量检索
    找到与 prompt 最相似的那一段内容
