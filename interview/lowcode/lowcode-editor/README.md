# Lowcode Editor

- npx create-vite lowcode-editor --template react-ts

- editor 组件
  - 低代码编辑器 
- tailwindcss 配置
   - 安装 tailwindcss 相关依赖   npm install tailwindcss @tailwindcss/vite
   - 配置 tailwindcss 相关文件

- allotment 组件
   pnpm i allotment 
   allotment 是一个用于在 React 应用中实现可拖拽分割面板（如左右 / 上下可调整大小的布局）的依赖包
   - 左侧：物料面板
   - 中间：编辑区域
   - 右侧：配置面板
- 模块化开发

- zustand 状态管理
   - json 数据，低代码编辑的本质
