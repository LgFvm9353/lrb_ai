# var_let_const

## JS代码的执行机制
- 有一段代码
  硬盘读入内存 
- V8引擎
  chrome 心脏，负责解析和执行代码
- 编译阶段
  - 代码执行环境
    currentVariable{
        showName: '',
        myName: ''
    }

  - 作用域 ： 变量查找的规则
     全局
     函数
     块级
   - 作用域链 ：变量查找的路径
     从当前作用域开始查找，没有就向上一级查找，直到找到为止，或者到全局作用域为止。
   嵌套关系 
   冒泡查找

- 变量提升 hoisting
  - 有作用域
  - var 提升变量的申明 undefined
    赋值是在执行阶段做
    但是函数提升是在编译阶段做的
    函数提升优先级高于变量提升
