关于 VibeStory

VibeStory 是一个关键词驱动的互动小说平台，通过简单的关键词组合，创造出丰富多彩的故事世界。

主要特点

  • 关键词驱动：只需输入几个关键词，即可生成完整故事
  • 互动选择：在故事的关键节点做出选择，影响故事走向
  • 角色关系：查看角色之间的关系网络
  • 故事结构：了解故事的叙事结构和关键节点
  • 记忆碎片：收集故事中的重要记忆，解锁隐藏内容

技术支持

本项目使用 DeepSeek API 提供故事生成服务，支持本地模式、代理模式和自定义代理模式。
